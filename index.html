Recent Projects

VisionAir

VisionAir


VisionAir Demo
VisionAir aims to provide its users the quality of the air around them, as opposed to getting the data from stations which may be many kilometers away, all while preserving the user’s privacy, which is another pressing issue in current times.


Key Features:


  • Uses the concept of Federated Learning to constantly grow and adapt to changes, while ensuring that the user’s data stays safe and in their control.

  • Uses a model trained on a dataset of 6000+ images taken across 80+ locations with more than 15 popular smartphones.

  • Nullifies any post-processing applied by the device’s camera, resulting in similar results for a scene even when using different devices. In other words, VisionAir implements both Scene Generalization and Hardware Generalization.

The application is currently available at the Google Play Store in Beta version. It is currently being tested in Delhi, and will soon expand to other areas.

Play Store Link: VisionAir

Reflecto-Mate

Reflecto-Mate

An Android Application that employs a chatbot to monitor a user’s stress level and counter depression. It merges standard psychological tests with casual conversation (using DialogFlow) to evaluate a person’s mental state, detect symptoms of and combat depression.


Key Features:


  • Merges psychological tests in general conversation to avoid the feeling of being interrogated.

  • Summarizes a user's statistics which can be viewed in the 'Dashboard'.

  • Gives the user the ability to doodle on a DrawBoard which may be then processed and evaluated for symptoms.

  • Aims to combat symptoms of depression by recommending exercises such as Breathing Techniques, or by recommending music and videos.

  • Uses rewards to motivate the user to try out various exercises.

The application is currently in development. The code may be found at my GitHub repository here: Reflecto-Mate GitHub Repository

FeDL

FeDL


On-Device Training for Linear Regressor (2X + 3) and Federated Server Testing
FeDL is an Android Application that implements On-Device Training and the concept of Federated Learning.

Federated Learning aggregates the improvements in the local models of various clients to update the server global model which is then pushed to the client devices.


Key Features:


  • Implements the concept of Federated Learning which can be used to constantly grow and adapt to changes, while ensuring that the user’s data stays safe and in their control.

  • Uses TensorFlow's Java API to implement On-Device Training.

  • Fetches weight files from the device and uploads them to the server for aggregation.

  • Pings the server and fetches the updated Global Model


The code may be found at my GitHub repository here: FeDL GitHub Repository

To learn more about Federated Learning, check out this Medium Article I wrote for VisionAir
